Slack Broadcaster

Send a message to multiple channels at once

First-time setup — get your Slack Bot Token:
  1. Go to api.slack.com/apps → Create New App → From scratch
  2. Under OAuth & Permissions → add scopes: chat:write, files:write, channels:read
  3. Click Install to Workspace → copy the Bot User OAuth Token (starts with xoxb-)
  4. Invite the bot to each channel: /invite @YourBotName
